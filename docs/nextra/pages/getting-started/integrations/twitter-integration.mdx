### Data tracked

Activities

* New followers
* New tweets matching a search query, for example, a Twitter handle (@GitMeshCE), a hashtag (#GitMeshCE), or a keyword (gitmesh.ce)

Contact attributes

* Name
* Profile picture
* Bio
* Location
* Follower count ("Reach")
