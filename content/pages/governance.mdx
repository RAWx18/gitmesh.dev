# GitMesh Community Governance

This section outlines the governance structure for GitMesh Community Edition, which operates under the Linux Foundation Decentralized Trust (LFDT).

## Project Structure

GitMesh Community Edition is maintained as an open-source project with the following structure:

- **Maintainers**: Core team responsible for project direction
- **Contributors**: Community members who contribute code, documentation, and feedback
- **Users**: Community members who use and provide feedback on the project

## Decision Making Process

1. **Technical Decisions**: Made by maintainers through consensus
2. **Community Input**: Gathered through GitHub discussions and issues
3. **Transparency**: All decisions documented in public repositories

## Contributing Guidelines

All contributions are welcome! Please see our contributing guidelines in the main repository for details on how to get involved.

## Code of Conduct

We follow the Linux Foundation Code of Conduct to ensure a welcoming and inclusive community for all participants.
